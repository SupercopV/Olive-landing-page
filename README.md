## ✅ Sections Implemented

- **Navbar** — Sticky navigation with logo, links, Sign in & CTA button
- **Hero** — Trust badge with avatars, headline, subtext, CTA buttons, floating phone mockup with live scan UI
- **How It Works** — 3-step cards with images
- **Health Benefits** — Dark green section with image cards and stat badges
- **Testimonials** — Featured testimonial + 2-card grid
- **Comparison Table** — Olive vs Others vs Manual
- **Pricing** — Free and Pro plan cards
- **Protect Banner** — Full-width image section with overlay
- **100% Independent** — Brand statement section
- **FAQ** — Interactive accordion
- **CTA Section** — Download prompt with feature list
- **Footer** — Multi-column navigation footer

## ✨ Features

- 📱 Fully responsive (mobile + desktop)
- 🎞️ Scroll-reveal animations using IntersectionObserver
- 🌊 Floating phone mockup animation (CSS keyframes)
- 🪗 Interactive FAQ accordion
- 🖼️ Real photography via Unsplash CDN
- ♿ Semantic HTML for accessibility
